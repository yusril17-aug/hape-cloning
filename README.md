# YSRL.Studio | Digital Experience 🪐

Building immersive digital worlds and future-ready products in Web3 culture. **YSRL.Studio** is a high-performance, scroll-sequenced 3D portfolio designed with a focus on "Digital Architecture" and premium user experience.

---

## 🏗️ Architectural Vision
This project is not just a website; it's a digital dimension. We merge cutting-edge WebGL engineering with sophisticated typography to redefine how users interact with the web.

## 🚀 Key Features
- **Scroll-Sequenced 3D Portfolio**: A seamless journey through WebGL environments powered by GSAP and React Three Fiber.
- **Digital Architecture Design**: A minimalist, industrial aesthetic using a custom design system (Syne & Inter typography).
- **High-Performance WebGL**: Optimized 3D meshes (Hero Knot) and environment designs (Stars, cosmic dust) for stable 60fps performance.
- **Custom Magnetic Cursor**: A reactive, fluid navigation experience that enhances user engagement.
- **Zero Re-render Architecture**: Advanced React optimizations for maximum smoothness during complex animations.

## 🛠️ Tech Stack
- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **3D Engine**: [React Three Fiber](https://r3f.docs.pmnd.rs/) & [Three.js](https://threejs.org/)
- **Animation**: [GSAP](https://gsap.com/) (ScrollTrigger, Custom Hooks)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Typography**: Syne (Display) & Inter (UI/Sans)

## 📦 Getting Started

### Prerequisites
- Node.js (Latest LTS)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/hape-cloning.git
   ```
2. Navigate to the directory:
   ```bash
   cd hape-cloning
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Run the development server:
   ```bash
   npm run dev
   ```
